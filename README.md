# IOT_03_Sensor-deUmidade 
Atividade desenvolvida com ESP32 como prática do componente para detectar a umidade do solo de uma planta na disciplina de IOT do curso Desenvolvimento de Sistemas do Senai

Segue um vídeo do código sendo executado em circuito:
https://www.youtube.com/watch?v=7wIxV6-jFUY

## Tecnologias utilizadas
* C++
* Arduino IDE
* Wokwi
* ESP32
* Espada de São Jorge e Iresine herbstii
