## Sobre
Projeto desenvolvido na disciplina de IIoT, ministrada pela professora [Yuri Komuta](https://www.linkedin.com/in/yurikomuta/), com o objetivo de implementar um sensor de umidade do solo.

## Descrição
A atividade consistiu no desenvolvimento de um sensor de umidade do solo. Para a realização do teste, foram utilizadas duas plantas: **Espada-de-São-Jorge** (solo seco) e **Iresine herbstii** (solo úmido).

Com base na leitura do sensor, um LED acende ou apaga dependendo da condição do solo. Além disso, o terminal exibe uma mensagem informando se o solo está **úmido** ou **seco**.
